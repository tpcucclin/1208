###  3-19 簡單回歸

```math
y=b_0+b_1x
```

假設：
```math
\bar{x}=8.3, \bar{y}= 54.8, SS_{x}=5.6, SS_{xy}=-12.4, SS_{y}=38.7
```
請估計 x = 5 時,y 的期望值

先求得 $b_0$
```math
b_0 = \bar{y} - b_1\bar{x}=54.8-(-2.21*8.3)=73.143 
```

再求  $b_1$ 
```math
b_1=\frac{SS_{xy}}{SS_{xx}} 
```

```math
b_1 = \frac{-12.4}{5.6}=-2.21 
```

當 x = 5 時,y 的期望值

```math
y=73.143-2.21x = 73.143-2.21*5=62.093
```
